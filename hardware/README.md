# [sudo] — Hardware

Open-source hardware for the sudo macro pad.

## Structure

```
hardware/
├── firmware/    # RP2040 firmware source
├── pcb/         # KiCad schematics & board files
├── case/        # Enclosure design files (STL, STEP)
└── docs/        # Build guide, BOM, wiring diagrams
```

## Specs

- **MCU**: RP2040
- **Buttons**: 4 mechanical keys
- **Output**: USB HID (Ctrl+Shift+F13–F16)
- **Connection**: USB-C
